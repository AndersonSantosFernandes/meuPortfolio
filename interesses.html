Meus interesses

HTML é uma linguagem de marcação utilizada na construção de páginas na Web.

Cascading Style Sheets é um mecanismo para adicionar estilo a um documento web.

JavaScript é uma linguagem de programação interpretada estruturada

PHP é uma linguagem interpretada livre, usada originalmente apenas para o desenvolvimento de aplicações presentes e atuantes no lado do servidor,

O MySQL é um sistema de gerenciamento de banco de dados, que utiliza a linguagem SQL como interface.

Quando comecei na programação eu estava meio sem um norte pra me encontrar, mas quando chegueo no terceiro semestre da faculdade tive o promeiro contato com o PHP.

Foi onde decidi focar meus estudos, pois com esta ferramenta junto com o HTML, JS e MySQL já é possível criar uma aplicação web de qualidade na modalidade fullStack.

Eu particularmente prefiro trabalhar com os dois lados da aplicação e não com um ou outro.
